DavronMarket Profile Header Plugin
=================================

Описание:
- Показывает шапку профиля пользователя на всех страницах сайта.
- Для авторизованных: аватар, имя, роль, ссылка на профиль.
- Для гостей: дефолтный аватар, "Гость", приглашение войти.
- Адаптивно для мобильных и десктопа.

Установка:
1. Подключите файл davron-profile-header.js на всех страницах сайта (перед </body>):
   <script src="/plugin/davron-profile-header/davron-profile-header.js"></script>
2. Для корректной работы требуется window.dmEngine.auth.currentUser().
3. Для аватара по умолчанию используйте /assets/img/default-avatar.png (или замените путь в коде).

Настройки:
- Нет дополнительных настроек. Для кастомизации — измените стили или HTML в плагине.
